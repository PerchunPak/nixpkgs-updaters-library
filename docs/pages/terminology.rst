Terminology
===========

.. glossary::

   Library
      This package.

   Updater script
      This is what you would run to add/update plugins in specific ecosystems.

   Input
      List of references to the outputs. Example: for Vim plugins, this would be
      a CSV table with Git URLs.

   Output
      What is generated at the end. This can be for example a plugin or an extension.

   Entry
      A single unit of what updater updates. For Vim plugins this would be
      a plugin, for Gnome extensions this would be an extension.

   ``G`` prefix
      Type is a `Generic <https://typing.python.org/en/latest/reference/generics.html>`_.
